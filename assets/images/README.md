Place your profile photo here as **profile.jpg** (a square image works best).

It is referenced from `index.md` and `about.md` as:

    profile: /assets/images/profile.jpg

You already have this file in your current repository — copy it into this folder
when you merge, or drop a new one in. If the file is missing the page still works,
the photo area just shows the image's alt text.
